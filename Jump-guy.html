Jump Guy

Unity 2D platform game

The Jump Guy was my first 2d mini game developed in unity during my first month in the bootcamp.

The game has a player controller wich allow the player move using the awsd or keyboard arrows. Also the player can jump using the space bar. It also, has the code necessary to detect collision with the obstacle and when the player collect coins.

I create a script to for generating new platforms, obstacles, and pickups as the player progresses. Finally I integrate a mini menu to start and restart the game as needed.

You can play this game on PC here: https://axlmonreal.com/jump-guy/
